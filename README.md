
# Expense Tracker

A simple and visually appealing **Expense Tracker** application built using **HTML, CSS, and JavaScript**.
This app allows users to add their daily expenses, categorize them, and view them in a clean, responsive table.

---

##  Features

* **Add Expenses**: Enter an expense name, amount, and select a category.
* **Auto-Display Table**: Newly added expenses instantly appear in the table.
* **Input Validation**: Alerts the user if fields are left empty.
* **Clean UI**: Minimal and soft-themed interface with a pastel gradient background.
* **Responsive Layout**: Works smoothly on all screen sizes.

---

##  Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling, layout, and animations
* **JavaScript** – Dynamic expense insertion and validation

---

##  How to Run

1. Clone or download this repository.
2. Open the `expensetracker.html` file in any web browser.
3. Start adding your expenses!
